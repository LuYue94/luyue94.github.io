    配置

    var scssFiles = ["src/scss/**/*.scss"];
gulp.task("sass",function(){
    gulp.src(scssFiles)
    .pipe(sass({outputStyle:"compact"}).on("error",sass.logError))    // 当sass文件编译报错时 不会停止 监听服务，错误部分不编译执行
    .pipe(gulp.dest("public/css"));
});

    输出格式

    outputStyle有以下4种选择：
    nested：嵌套缩进，它是默认值
    expanded：每个属性占一行
    compact：每条样式占一行
    compressed：整个压缩成一行
